How to HTML/CSS/JS

HTML

Headings

They are used to define the structure of the page. There are 6 levels of headings in HTML, from <h1> to <h6>. The <h1> tag is used for the main heading of the page, and the other tags are used for subheadings.

Paragraphs

They are used to define blocks of text on the page. The <p> tag is used to create a paragraph. Paragraphs are used to group related content together and make the text more readable.

Lists

There are two types of lists in HTML:

  • Unordered lists are used to create a list of items that do not have a specific order. The <ul> tag is used to create an unordered list, and the <li> tag is used to create a list item.
  • Ordered lists are used to create a list of items that have a specific order. The <ol> tag is used to create an ordered list, and the <li> tag is used to create a list item.

Links

You create a link on the page by creating an "anchor" tag. The <a> tag is used to create a link. The "href" attribute specifies the URL of the link, and the "target" attribute specifies where to open the link. In this case, the "target" attribute is set to "_blank", which means the link will open in a new tab or window.

HTML
HTML is the "structure" of a web page. It defines the elements that make up the page, such as headings, paragraphs, images, and links. HTML is the foundation of web development.
CSS
CSS is the "style" of a web page. It defines how the HTML elements should look, such as colors, fonts, and layout. CSS is used to make web pages visually appealing.
JavaScript
JavaScript is the "behavior" of a web page. It allows you to add interactivity and dynamic content to your web pages. You can use JavaScript to respond to user actions, manipulate the HTML and CSS, and create complex web applications.
